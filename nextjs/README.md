# nextjs-playground

🚀 **Next.js Playground** is a personal learning repository where I explore and practice core and advanced concepts of **Next.js**.

This repo is focused on understanding how Next.js works under the hood, including:
- File-based routing and dynamic routes
- Server Components and Client Components
- Static Site Generation (SSG)
- Incremental Static Regeneration (ISR)
- Server-Side Rendering (SSR)
- Data fetching patterns
- Navigation and layouts
- Rendering strategies and performance

Each folder or example represents a small experiment or concept-based implementation to build a strong foundation in Next.js.

📌 **Purpose**:  
To learn Next.js deeply by experimenting, breaking things, fixing them, and understanding *why* things work — not just *how*.

🧠 This repository is for **learning, practice, and reference**, not a production-ready application.
