## 🧭 Learning Flow

I followed a step-by-step path to build strong frontend fundamentals:

1. 🌐 **HTML**
   - Structure of web pages
   - Semantic elements

2. 🎨 **CSS**
   - Flexbox & Grid
   - Responsive design

3. 🅱️ **Bootstrap**
   - Pre-built UI components
   - Rapid layout building

4. 🧩 **SCSS**
   - Variables, Nesting, Mixins
   - Scalable styling

5. ⚡ **JavaScript**
   - Core concepts (closures, promises)
   - DOM manipulation

6. 💨 **Tailwind CSS**
   - Utility-first styling
   - Fast UI development

7. ⚛ **React.js**
   - Components & Hooks
   - State management
   - Mini projects

8. ▲ **Next.js**
   - Routing & SSR basics
   - Production-ready apps

9. 🟦 **TypeScript**
   - Type safety
   - Better scalability in React
